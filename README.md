Repository ini merupakan repository penugasan tes centrinova yang berisikan website front end artikel bernama Blogu

## Link
[https://centrinova-blogu.vercel.app/](https://centrinova-blogu.vercel.app/)

## Dependency
- vue (2.6.11)
- vue-router (3.2.0)
- vuetify (2.4.0)
- axios (0.21.1)
- dayjs (1.10.6)
